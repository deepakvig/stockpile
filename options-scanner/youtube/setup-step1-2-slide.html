Setup
getting the code and Python on your machine
Step 1
Get the code
Download the zip or clone from GitHub:
git clone https://github.com/medloh/stockpile.git
cd stockpile
Needs Git installed — git-scm.com has installers for Windows and Mac. Or just download the zip from the repo page.
Step 2
Install Python 3.12+
Download the installer for your platform:
python.org
Windows: check "Add Python to PATH" during installation — that's the one people miss.
